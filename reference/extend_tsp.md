# Extend tsp with lf

This window is the smallest that is all around tsphf that is compatible
with the low frequency.

## Usage

``` r
extend_tsp(tsphf, lffreq)
```

## Arguments

- tsphf:

  a numeric of length 3, a tsp of high-frequency

- lffreq:

  a numeric of length 1, the low frequency

## Value

a numeric of length 3, a tsp of high-frequency.
